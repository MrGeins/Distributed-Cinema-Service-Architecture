# Distributed Cinema Service Architecture
Engineered a multi-tier web platform orchestrating seamless communication between a robust Spring Boot backend for complex data aggregation and Node.js microservices. Implemented Socket.IO for real-time concurrency and RESTful patterns to ensure efficient data synchronization across heterogeneous server environments.
Containerized the entire ecosystem using Docker and Docker Compose, ensuring consistent deployment across development and testing environments.
